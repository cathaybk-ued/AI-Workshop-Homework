# AI Workshop Homework

AI 工作坊學員作業繳交區（試教場 + 正式場）。

**作業一律以 `.html` 格式繳交。** 請放入對應資料夾，並依照下方命名規則命名檔案。

---

## 一、資料夾結構

```
AI-Workshop-Homework/
├── 試教場/
│   └── 中文+英文名字/                # 例：王小明 Ming
│       └── xxxx.html
└── 正式場/
    ├── 個人日週報 Agent/
    │   └── 中文+英文名字/            # 例：王小明 Ming
    │       └── 0714_weekly_report_Ming.html
    └── 自定義 Agent/
        └── 中文+英文名字/            # 例：王小明 Ming
            └── 0714_MyAgent_Ming.html
```

- 正式場請放到對應的子資料夾：**個人日週報 Agent** 或 **自定義 Agent**。
- 每位學員在資料夾內用**自己的「中文+英文名字」建一個子資料夾**（例：`王小明Ming`），作業放進去。

---

## 二、檔案命名格式

副檔名一律 `.html`。四碼日期為 `MMDD`（例：7 月 14 日 = `0714`）。

### 個人日週報 Agent

```
四碼日期_weekly_report_英文名字.html
```

範例：`0714_weekly_report_Ming.html`

### 自定義 Agent

```
四碼日期_自己agent名字_英文名字.html
```

範例：`0714_MyAgent_Ming.html`

規則：
- 欄位之間一律用底線 `_` 分隔，欄位內不要有空格。
- 檔名請勿使用特殊符號（`/ \ : * ? " < > |`）。

---

## 三、如何上傳

### 方式 A：GitHub 網頁上傳（已被加為協作者）

1. 進入對應資料夾。
2. 點右上角 **Add file → Upload files**（可直接拖曳資料夾建立子資料夾）。
3. 拖曳你的 `.html` 作業檔案進去。
4. 下方 **Commit changes** 直接送出。

> 建立子資料夾小技巧：**Add file → Create new file**，檔名輸入 `王小明 Ming/0714_weekly_report_Ming.html` 即可自動建立資料夾。

### 方式 B：Fork + Pull Request（一般學員，推薦）

1. 點右上角 **Fork**，複製一份到自己的帳號。
2. 在自己的 fork 中，進入對應資料夾上傳作業檔案（同方式 A 步驟 2–4）。
3. 回到本專案，點 **Pull requests → New pull request**，選你的 fork 分支送出。
4. 講師 review 後合併。

> 沒有 GitHub 帳號者，請先至 https://github.com 免費註冊。
